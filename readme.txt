Description: The package called "dataset" is to transform the format of the data in order to run the code more conveniently. A python 3.5 implementation of MLDF proposed in Multi-Label Learning with Deep Forest. 

References:L. Yang, X.-Z. Wu, Y. Jiang, and Z.-H. Zhou. Multi-label learning with deep forest. In: Proceedings of the 24th European Conference on Artificial Intelligence (ECAI'20), Santiago de Compostela, Spain, 2020.

Requirement: This package is developed with Python 3.5.

ATTN: This package is free for academic usage. You can run it at your own risk. For other purposes, please contact Prof. Zhi-Hua Zhou (zhouzh@nju.edu.cn).
